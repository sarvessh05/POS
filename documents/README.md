## 🚀 Quick Start for Developers

If you are a new developer onboarding onto the project, please follow this reading order:

1. **Read the Architecture Document** to understand how the apps (Admin, Captain, Customer) interact with Supabase.
2. **Review the Code Documentation** to familiarize yourself with the directory structure and custom hooks.
3. **Check the API Documentation** to see how data is fetched and mutated.
4. **Run the system locally** as described in the root `README.md`.
